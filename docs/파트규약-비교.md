# 파트별 규약 비교표 & 통합 로드 규격 (2주차 산출물)

> 대상: 2주차에 로드한 4개 파트(ⓐ모바일·ⓑ로봇팔·ⓒ엔드이펙터·ⓓ센서).
> 목적: 파트별 URDF 네이밍·좌표·DOF 규약을 비교해 **충돌 지점**을 찾고, 통합 URDF의 **공통 로드 규격**을 정한다.
> 근거: 각 파트 xacro→urdf 실측(`check_urdf` 통과분) 기준. 작성일 2026-07-14.

## 2. 통합 시 충돌·주의 지점

1. **`base_link` 이름 충돌** — Scout·D435i·D405가 모두 `base_link` 사용.
   → 해결: Scout `base_link`를 **전체 루트**로 채택. 센서는 매크로 `name:=` 인자로 프레임을 `d435i_*`/`d405_*`로 네임스페이스(매크로가 base_link를 만들지 않고 `<name>_link`를 parent 아래 생성).
2. **RB5 무접두 링크(`link0..6`)** — 다른 팔/파트 추가 시 충돌 위험, 의미 불명확.
   → 현실: `rb_6dof` 매크로의 `prefix`는 **버그로 안전하지 않음** — `link`/`joint` 서브매크로가 관절 parent/child를 `joint.yaml`에서 **무접두**(`link0`,`link1`…)로 읽어오므로, `prefix='arm_'`를 주면 링크는 `arm_link0`인데 관절은 `link0`을 참조해 트리가 끊긴다.
   → 초안 결정: **`prefix=''` 사용**(`link0..6`,`tcp` 유지). 현재 다른 파트(Scout `base_link`, RG2 `rg2_*`, 센서 `d405/d435i_*`)와 실제 이름 충돌 없음.
   → 향후 과제: 다중 팔/네임스페이스 필요 시 `rb_6dof`+`joint.yaml`을 prefix-aware 하게 개조하거나 URDF 후처리로 리네임.
3. **Scout가 매크로가 아님** — prefix 불가. 유일하게 루트여야만 자연스럽게 편입됨.
   → 해결: Scout를 루트로 고정. (향후 다중 로봇/네임스페이스가 필요하면 Scout xacro의 매크로화가 별도 과제.)
4. **RG2 mimic 원본 오기**(panda_finger_joint1) — 편입판에서 `rg2_finger_joint1`로 수정 완료.
5. **좌표/장착 변환은 임시값** — 각 파트 간 실제 장착 위치(플랜지 볼트패턴·마스트 높이)는 도면 확정 전까지 초안 추정값. 통합 URDF의 mount origin은 TODO로 표기.

## 3. 통합 로드 공통 규격 (초안)

- **루트 프레임**: `base_link`(Scout) — 지면 접촉 `base_footprint` 보유.
- **네임스페이스 규약**:
  - 로봇팔 → 무접두 `link0..6`,`tcp` (매크로 prefix 버그로 초안은 무접두. 충돌 없음)
  - 그리퍼 → 무접두 `rg2_*` (macro prefix 지원하나 초안은 무접두 유지)
  - eye-in-hand 센서 → `d405` (손목/그리퍼 장착)
  - eye-to-hand 센서 → `d435i` (베이스/마스트 장착)
- **조립 체인**:
  ```
  base_link (Scout)
    ├─ [fixed] arm_mount_joint → link0 (RB5, prefix='')
    │     └─ … link6 → tcp
    │            └─ [fixed] gripper_mount_joint → rg2_hand (RG2)
    │                   └─ [fixed] d405 부착 → d405_link (eye-in-hand)
    └─ [fixed] mast_joint → d435i_link (eye-to-hand)
  ```
- **단위**: SI (m, rad) — 4파트 모두 일치, 변환 불필요.
- **표시 검증 방법**: 수동 스택(robot_state_publisher `urdf파일` + joint_state_publisher_gui + rviz2 `-d config`)이 벤더 launch 경로다툼보다 안정적.

*통합 URDF 초안 파일: `rda_robot_description/urdf/rda_robot.urdf.xacro`*
