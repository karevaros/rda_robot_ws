# AIHub 토마토 데이터 → 작물 파라미터 근거 (집계 수치)

출처: **AI-Hub 「지능형 스마트팜 통합 데이터(토마토)」**(dataSetSn=534),
Training 라벨 62,301개 집계(2026-07-20). 원본 데이터는 재배포 제약·대용량이라
저장소에 포함하지 않음(`.gitignore`) — 아래 **파생 통계**만 근거로 남긴다.

성격: 이 데이터셋은 **생육 진단용**(2D 이미지 + LabelMe 바운딩박스/폴리곤 + 생리 측정값).
3D 좌표·깊이·카메라 캘리브레이션이 없어 **공간 배치나 픽셀→cm 변환은 불가**. 아래 표의
"직접 사용"만 기하 파라미터로 채택했다.
